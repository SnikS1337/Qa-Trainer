# QA Trainer v2.0

Интерактивный тренажер для начинающих тестировщиков (QA Manual). Обучение в игровом формате, вдохновленном Duolingo.

## Особенности
- **Игровой процесс:** Набирайте XP, повышайте уровень и открывайте новые уроки.
- **Разнообразные задания:** Выбор одного варианта, сортировка, практические задачи.
- **Система достижений:** 12 уникальных ачивок за различные успехи.
- **Ежедневные квизы:** Поддерживайте серию (streak), отвечая на вопросы каждый день.
- **Экзамен:** Проверьте свои знания в финальном тесте и получите сертификат.
- **Современный дизайн:** Интерфейс с эффектом стекла и плавной анимацией.

## Назначение приложения
QA Trainer - это обучающее приложение, предназначенное для начинающих специалистов по тестированию программного обеспечения. Оно помогает освоить основы QA в интерактивной и увлекательной форме, позволяя пользователям изучать ключевые концепции тестирования, практиковаться в решении задач и проверять свои знания через викторины и экзамен. Приложение идеально подходит как для самообучения, так и для использования в образовательных целях.

## Скрытые функции (Debug)
В приложении есть скрытые возможности для отладки и тестирования:

1. **Dev Menu (Меню разработчика):**
   - Чтобы открыть, **удерживайте иконку колбы (🧪)** в шапке главного экрана в течение 1 секунды.
   - Потребуется пароль: `trainer`.
   - Возможности: добавление XP, разблокировка всех уроков, экспорт/импорт сохранений, полный сброс.
   - *Внимание:* Использование Dev Menu помечает профиль как "читерский", что блокирует официальную выдачу сертификата.

2. **Debug Certificate (Просмотр сертификата):**
   - Если сертификат заблокирован (не пройдены уроки или использованы читы), вы можете просмотреть его для отладки.
   - На экране сертификата **удерживайте иконку шляпы (🎓)** в течение 3 секунд.
   - Потребуется пароль: `cert`.
   - Это разблокирует предпросмотр сертификата, но кнопка скачивания останется недоступной и ввод имени будет заблокирован, если профиль помечен как "читерский" или использовался дебаг-режим.

## Безопасность и честность
- Приложение отслеживает использование Dev Menu и помечает профиль как "читерский"
- Сертификат не выдается читерам, даже при прохождении всех уроков
- Система предотвращает обход правил получения сертификата

## Развертывание на GitHub Pages
Проект настроен для автоматического развертывания через GitHub Actions. 

**Важно:** В `vite.config.ts` используется `base: './'`, чтобы собранное приложение корректно работало и локально, и при публикации в подпуть GitHub Pages без ручной правки путей к ассетам.

## Установка и запуск
1. Установите зависимости: `npm install`
2. Запустите в режиме разработки: `npm run dev`
3. Соберите проект: `npm run build`
