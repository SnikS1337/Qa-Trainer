# QA Trainer

Интерактивный тренажер по ручному тестированию для начинающих QA-инженеров. Проект сочетает уроки, практические задания, ежедневный квиз, экзамен, достижения и сертификат в одном офлайн-first SPA.

## Что есть в приложении
- Последовательное прохождение уроков с XP, уровнями и разблокировкой контента.
- Несколько типов заданий: single choice, сортировка, практические сценарии и финальный экзамен.
- Ежедневный квиз со стриком и локальной историей прогресса.
- Система достижений, статистика, сертификат и защитные ограничения для дебаг-режимов.
- UI с анимациями, glassmorphism-слоями и кастомными состояниями карточек.

## Стек
- React 19
- TypeScript
- Vite 6
- Tailwind CSS v4 (`@tailwindcss/vite`)
- Motion
- Canvas Confetti

## Локальный запуск
```bash
npm install
npm run dev
```

Приложение поднимается на `http://localhost:3000`.

## Скрипты
- `npm run dev` - локальная разработка
- `npm run lint` - проверка TypeScript через `tsc --noEmit`
- `npm run build` - production build Vite
- `npm run preview` - локальный просмотр production build
- `npm run clean` - удаление `dist`
- `npm run check` - полный локальный прогон `lint + build`

## Хранение данных
- Состояние пользователя хранится в `localStorage`.
- Основной store находится в `src/store.ts`.
- Приложение не использует backend и внешние API.

## Debug-функции
- `Dev Menu`: удержание иконки `🧪` на главном экране, пароль `trainer`
- `Cert Debug`: удержание иконки `🎓` на экране сертификата, пароль `cert`
- Использование debug-возможностей помечает профиль как читерский и блокирует официальную выдачу сертификата

## GitHub Pages и Actions
- Workflow находится в `.github/workflows/deploy.yml`.
- Для `pull_request` в `main` выполняется проверка проекта через `npm run check`.
- Для `push` в `main` после проверки запускается деплой на GitHub Pages.
- В `vite.config.ts` используется `base: './'`, чтобы сборка работала и локально, и в подпути Pages.

## Структура проекта
```text
src/
  components/   переиспользуемые UI-компоненты и модалки
  data/         части данных уроков
  screens/      экраны приложения
  App.tsx       переключение экранов, фон, toast, root layout
  data.ts       уроки, практика, достижения, цитаты и доменные данные
  store.ts      глобальное состояние и localStorage
  types.ts      основные типы приложения
  utils.ts      вспомогательные функции
  index.css     глобальные стили и утилиты
```
